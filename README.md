# Landmacht Veiligheid Marketing Website

Production-ready Next.js marketing website for Landmacht Veiligheid with App Router, Tailwind CSS, MDX insights, service pages, and local SEO focus for Franschhoek, Paarl, and Wellington.

## Run Locally

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run start
```

## Image Assets Note

Store all production image files in `/public/images`. The app uses `/images/placeholder.jpg` as a temporary fallback when an expected image file is missing.
